Grid Js

  1. Tables
  2. Grid Js
OverviewOfficial Website

Grid.js is a Free and open-source table plugin. It works with most JavaScript frameworks.

Import package
<!-- gridjs -->
import { Grid, _ } from 'gridjs-react';
Add Package
yarn add gridjs-react --save
Remove Package
yarn remove gridjs-react or you can remove package by removing specific package from package.json

Base Example 
<Grid
    data={data}
    columns={[{
    name: 'ID',
    formatter: (cell) => _(<span className="fw-semibold">{cell}</span>)
    },
    "Name",
    {
    name: 'Email',
    formatter: (cell) => _(<a href="/#"> {cell} </a>)
    },
    "Position", "Company", "Country",
    {
    name: 'Actions',
    width: '120px',
    formatter: (cell) => _(<a href='/#' className='text-reset text-decoration-underline'> Details </a>)
    },
    ]}
    search={true}
    sort={true}
    pagination={{ enabled: true, limit: 5, }}
/>
